Automatically shows your doc, xls and pdf attachments under every post and page. Simply and Easily. As it should be!

== Description ==

This plugin automatically shows your doc, xls and pdf attachments under every post and page of your website. A simple, fast and easy way to avoid having to insert your file links in the content.

== Installation ==
1. Install the plugin either via the WordPress.org plugin directory, or by uploading the files to your server
2. After activating, you've all done.
